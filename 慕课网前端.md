# HTML

设置网页**关键词**

```javascript
<meta name="Keywords" content="慕课网，imooc，Java，前端...">
```

设置网页**描述**

```javascript
<meta name="Description" content="慕课网是一个学习平台......">
```

title是搜索引擎收录网站时显示的标题，同时搜索引擎非常看重`<h1></h1>`标签的内容



| 转义字符 | 意义               |
| :------- | :----------------- |
| `&lt;`   | 小于号             |
| `&gt;`   | 大于号             |
| `&nbsp;` | 空格（不会被折叠） |
| `&copy;` | 版权符号@          |

原型图标注软件：Axure 或者 Sketch
